# AgentCanvas SaaS

## Project Structure
- `apps/api`: Fastify API & Rendering Engine (Backend)
- `apps/web`: Next.js Dashboard & Landing Page (Frontend)

## Quick Start

### Backend (API)
```bash
cd apps/api
npm install
npm run dev
```

### Frontend (Web)
```bash
cd apps/web
npm install
npm run dev
```

## Architecture
- **Backend:** Node.js, Fastify, BullMQ, Redis, Remotion
- **Frontend:** Next.js 14, Tailwind CSS, Shadcn/UI
